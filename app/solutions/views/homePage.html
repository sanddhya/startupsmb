About Pitney Bowes SendPro® C-Series

The SendPro® C-Series is the next generation office sending solution with an Android based Tablet. Developers can create apps for the tablet and web to improve and amplify productivity.

The SendPro® C-Series streamlines the mailing and shipping options for a consistent experience every single time a Pitney Bowes customer ships.

It is the simplest way to mail, ship and track consignments.

2. Install & Setup Android Studio.

Mac Users

  • Download the installer to your PC and launch the .dmg file from the Android Studio home page.

  • Drag and drop the .dmg into your Applications folder.

  • Open Android Studio and follow the instructions from the setup wizard.

  • If you get a warning saying that the file is damaged and should be moved to the trash,
    go to System Preferences > Security & Privacy and under the “Allow applications downloaded from” section select “Anywhere.”

  • Continue with wizard instructions

Windows Users

  • Install JDK 6.0 or above

  • Download and launch the .exe file on your PC

  • Follow the instructions in the setup wizard

  • Setup environment variable for Java

  • - Select Start menu > Computer > System Properties > Advanced System Properties.

  • - Open Advanced tab > Environment Variables and add a new system variable JAVA_HOME that points to your JDK folder, for example, C:\Program Files\Java\jdk1.7.0_21

Using Demo App

Import the Apps

You may use Android Studio or any other IDE to explore the demo Apps.
To import App in the Android Studio:
• Open Android Studio
• Click Open an existing Android Project
• Navigate to the folder where Demo App is downloaded and click OK.
- If you have GitHub account, you can directly import by using “checkout project from version control” and following the wizard
• Notice that Android Studio begins compiling. It also prompts to download additional SDK tools, as needed.
• Follow the readme document of demo App and add your API and Secret keys at the specified location.

Run the Emulator

Once the app is successfully compiled, you can run it using Android Studio Emulator. To do so:
• Launch the Emulator.
• Click Run app or press Ctrl+R.
• Select available virtual device or create a new device
• Notice a demo App running on Virtual Device.
Alternatively, you can create an APK file by clicking Build APK and install on the real Android device
