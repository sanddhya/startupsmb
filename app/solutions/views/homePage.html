How to use Startup Kit

1. Register and Claim API keys

  • Register for a Pitney Bowes Developer Account. No credit card required.

  • Login to the Developer Hub using your developer account credentials.

  • Find your API key and secret under locate tab on the top of the page.

  • You will need these keys for using Pitney Bowes Developer APIs.

2. Install & Setup Android Studio.

Mac Users

  • Download the installer to your PC and launch the .dmg file from the Android Studio home page.

  • Drag and drop the .dmg into your Applications folder.

  • Open Android Studio and follow the instructions from the setup wizard.

  • If you get a warning saying that the file is damaged and should be moved to the trash,
    go to System Preferences > Security & Privacy and under the “Allow applications downloaded from” section select “Anywhere.”

From here you can repeat Step 3 and install the program.

Windows Users

  • Download and launch the .exe file to your PC from the Android Studio home page.

  • Follow the instructions on the setup wizard to install Android Studio.

  • If asked to point to where Java is installed, you need to set an environment variable in order to direct the installer to the proper location

To do that, select Start menu > Computer > System Properties > Advanced System Properties. From there you’ll open the “Advanced” tab and click “Environment Variables.” Here you’ll add a new system variable titled JAVA_HOME that points to your JDK folder.

Import, Build & Run

Import Demo Apps

• You may use Android Studio or any other IDE of your choice to explore the demo Apps.
• Import App in Android Studio:
o Open Android Studio.
o Click on “Open an existing Android Project”
o Navigate to the folder where Demo App is downloaded and click “ok”.
o If you have GitHub account, you can directly import by using “checkout project from version control” and following the wizard.
o Once imported Android Studio will compile and may prompt for download additional SDK tools.
o Follow the readme document of demo App and add your API keys and secret at specified locations.

Build App and Run on emulator

• Once the app is successfully compiled you can run it using Android Studio Emulator (virtual device).
• Please follow link to know more about creating and managing Android Virtual device using Android Studio.
• Click on RunRun ‘app’ on the toolbar or press ctrl+R.
• Select available virtual device or create a new one.
• You should be able to see the Demo App running in Virtual Device.
• Alternatively, you may create an apk file by clicking BuildBuild APK and install on real Android device.
